# Description
this application consumes a contacts api. The contacts are listed and it is possible to view the details of each contact.

# Technologies
- React-Native (main components: FlatList, ActivityIndicator, TouchableOpacity)
- React-navigation (3.0.9)

# Api
- axios
- https://randomuser.me/api/?nat=br&results=150
